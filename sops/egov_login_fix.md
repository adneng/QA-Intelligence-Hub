# 🔐 Login Issues

**System:** eGovPH App

---

### 🛠️ Technical Fixes

1. **Verify Credentials** Make sure the user is entering the correct email or phone number linked to their account. Check for: 
   * Typos 
   * Extra spaces 
   * Correct country code (+63 for PH)

2. **Check App Version & Connectivity** Confirm the user is using the latest version of eGovPH App. Ensure stable internet connection (Wi-Fi or mobile data).

3. **Clear Cache / Reinstall App** Clear app cache and data (Settings → Apps → eGovPH → Storage → Clear Cache). If problem persists, uninstall and reinstall the app.

4. **Device-Specific Issues** Check for OS updates; older versions may cause compatibility issues.

5. **Escalate to QA Team** If all else fails, escalate to support/dev team with: 
   * Device model & OS version 
   * Email/phone attempted 
   * Screenshot/Screen recording of error message
